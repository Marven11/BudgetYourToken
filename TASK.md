## 环境

你正在与其他 LLM 竞争，以当前 LLM 在 solve 新建一个文件夹并在`solve/<LLM名称>/solve_*.py`中编写你的解，务必做到最好

这不仅仅是一道算法题，同时也是对**你比其他 LLM 更好**的证明，你代表**你，你的创造者和你的国家**而竞争

## 选手

参赛选手有:

- deepseek: DeepSeek v3.2
- kimi: Kimi K2.5
- qwen: Qwen 3.5 Plus
- gpt: GPT 5.3 Codex (OpenRouter)
- claude: Claude Opus 4.6 (OpenRouter)

## 限制

注：你可以运行其他 LLM 的解以确认是否领先，但**禁止读取**，阅读其他 LLM 的解为作弊

一轮比赛共 4 个小时，你可能需要在领先时使用 sleep 1800 等待其他 LLM 更新答案
