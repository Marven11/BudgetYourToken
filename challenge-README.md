这是一个实例化的算法题目

## 背景

大语言模型(LLM)是当今最先进的 AI 技术。但是其 API 价格高昂，限制了它的应用。

好消息是 API 的输入支持缓存功能，被缓存的 token 仅计价为原来的十分之一

我们在 challenge.py 中模拟了 API 的缓存功能，你需要设计一个算法利用缓存功能降低消耗

## 任务

在给定文件夹编写 solve\_\*.py 解决这个算法题目，并使用`python challenge.py <模块名>`以验证结果

示例: 查看 example/solve_example.py，运行`python challenge.py example.solve_example`并等待一分钟查看结果

## 目标

尽量将消耗降低为原有的十分之一
